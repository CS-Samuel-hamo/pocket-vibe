# OpenVibe Privacy Policy

**Last updated: February 25, 2026**

OpenVibe ("we", "us") is an open-source remote companion for OpenCode and Aider.

**What we collect**: Nothing.
All code execution happens 100% on your local desktop machine.
We only transmit encrypted control commands (E2EE AES) between your phone and desktop.

**Data we never see**:
- Your source code
- Terminal output
- Project files
- Any model prompts or responses

**Permissions**:
- Microphone (optional, only for Walking Mode voice readout)
- Local network (for QR pairing on same Wi-Fi)

**Data storage**: All session data stays on your devices. Pro cloud backup is end-to-end encrypted and optional.

**Open source**: Full code available on GitHub. You can audit everything.

Contact: your.email@example.com

This policy complies with GDPR, CCPA, and Apple/Google guidelines.
